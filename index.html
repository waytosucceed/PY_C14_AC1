Sets in Python
In Python, a set is an unordered collection of unique elements.

Characteristics of a set:

  • üîÄ Unordered: Sets do not maintain any order of elements.
  • üîß Mutable: You can add or remove elements from a set, but elements themselves must be immutable.

We often encounter sets in our daily lives.

Membership Operator with Sets
üîç The membership operator in Python helps us check if a specific value is present in a set (or other collections like lists or dictionaries).

üìå Do you remember the membership operator?
The membership operator is represented by the keyword in, and we use it to test if a value exists in a set.

üìä Example with sets: 
        superhumans_set = {'Iron Man', 'Spider-Man', 'Hulk', 'Thor'}
        print('Iron Man' in superhumans_set)  # Output: True
        print('Captain America' in superhumans_set)  # Output: False
In this example: - `'Iron Man' in superhumans_set` returns `True` because "Iron Man" is present in the set. - `'Captain America' in superhumans_set` returns `False` because "Captain America" is not in the set.

üìå Key Takeaways:
  • üîé Use the in operator to check if a value exists in a set.
  • ‚ùå If the value is not present, it will return False.
  • ‚úîÔ∏è If the value is found, it returns True.

Set Operations
In Python, you can perform various operations on sets using operators like &, |, -, <=, >=, and ^.

üìå Union (|)



This operator combines multiple sets. You can have more than 2 sets too!

üìå Intersection (&)



This operator gives a new set with elements common to all sets.

üìå Difference (-)
The difference operator returns a set of elements present in one set but not in the other.
‚óè list1 - list2 gives elements in list1 but not in list2.
‚óè list2 - list1 gives elements in list2 but not in list1.










üìå Symmetric Difference (^)


This operator gives a set of elements that are in either of the sets but not in both.

üìå Key Takeaways:
  • üîó Union (|) combines multiple sets into one.
  • üîÑ Intersection (&) finds common elements in sets.
  • ‚ûñ Difference (-) shows items present in one set but not another.
  • üîÄ Symmetric Difference (^) shows items that are unique to each set.
